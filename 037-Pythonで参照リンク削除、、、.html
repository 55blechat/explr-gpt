🧾 ステップ②：整形スクリプトの準備

以下をそのままメモ帳などにコピペして、
clean_md.py という名前で保存してください（拡張子は .py）

import re

def simplify_markdown_with_emojis_only(md_text: str) -> str:
    cleaned = re.sub(r'\[(.*?)\]\(https?://[^\)]+\)', r'\1', md_text)
    cleaned = re.sub(r'\n{3,}', '\n\n', cleaned)
    return cleaned

input_path = "chatgpt_log.md"
output_path = "chatgpt_log_cleaned.md"

with open(input_path, "r", encoding="utf-8") as infile:
    original_md = infile.read()

cleaned_md = simplify_markdown_with_emojis_only(original_md)

with open(output_path, "w", encoding="utf-8") as outfile:
    outfile.write(cleaned_md)

print(f"✅ 整形完了！ → {output_path}")

📁 ステップ③：Markdownファイルと同じフォルダに置く

  1. 上で作った clean_md.py を保存

  2. 同じフォルダに chatgpt_log.md（加工前のMarkdown）を入れる

  3. フォルダ内で Shift + 右クリック → PowerShell/コマンドプロンプトで開く

  4. 次のコマンドで実行：

python clean_md.py

すると chatgpt_log_cleaned.md が生成されます 🎉

def simplify_markdown_with_emojis_only(md_text: str) -> str:

    cleaned = re.sub(r'\n{3,}', '\n\n', cleaned)
    return cleaned

 実行時にユーザーからファイル名を聞く
input_path = input("整形したいMarkdownファイルの名前を入力してください（例: chat.md）: ").strip()

if not os.path.exists(input_path):

 出力ファイル名を自動生成（例: chat_cleaned.md）
output_path = input_path.replace(".md", "_cleaned.md")

 ファイル読み込み・整形・保存
with open(input_path, "r", encoding="utf-8") as infile:
