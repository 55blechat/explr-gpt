🔧 VS Codeってどんな人が使ってるの？

ジャンル 主な用途 利用者の例
プログラマー・エンジニア コーディング全般（HTML, Python, C++, etc） Web系、AI系、組み込み系、ゲーム開発者
データサイエンティスト PythonやRでの分析、Jupyter連携 機械学習、統計解析など
Markdown愛用者 ドキュメントやブログ執筆 noteやObsidianユーザー、テクニカルライター
学生・初学者 プログラミング入門、課題作成 大学生、専門学校生、プログラミングスクール生
IT以外の一般ユーザー メモ、文章整理、スクリプト補助 ライター、研究者、ノーコード利用者なども増加中

🌍 世界的な人気度（ちょっと驚くかも）

  • Stack Overflow（世界最大の開発者調査）で数年連続「最も使われている開発環境」

  • GitHubなどでの推奨環境にも指定されがち

  • Google Docs や Word より軽くて整ったMarkdown編集ができることから文章作成ツールとしても評価急上昇中

💼 ビジネスツールとしての認知度は？

✅ IT・開発系の企業では完全に「常識レベル」

  • 新人研修では必ず使う

  • 社内ドキュメントもMarkdownで書いて、VS Codeで編集することが一般的

  • 一般的なオフィス業務（営業・事務・広報など）では、まだそこまで浸透していない

  • Word / Excel が主流の現場では、「VS Code？なにそれ？」となることも多い

✅ なぜ一般人にもおすすめされ始めてるの？

  • 軽い・無料・多機能

  • Markdown・Python・HTMLなど複数用途に使える

  • 拡張機能で「自分専用ツール」に育てられる

  • デザインやUIが分かりやすい

つまり、「知る人ぞ知る便利ツール」から「知った人から順に手放せなくなる系ツール」へと広がっている感じです。

まとめると

あなたのように「文章＋構造＋整理＋再利用」をしたい人には、VS Codeは最強のパートナーです。
