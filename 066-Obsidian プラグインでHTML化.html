✅ 方法①：ObsidianプラグインでHTML化（簡単＆Obsidian内で完結）

🔧 プラグイン名：Advanced Export（旧 Obsidian Export）

📦 インストール手順：

    1. Obsidian → ⚙「設定」→「コミュニティプラグイン」→「Browse」

    2. 検索欄に Advanced Export と入力

    3. インストール → 有効化（Enable）

🧰 使い方：

    1. .md ファイルを開く

    2. コマンドパレット（Ctrl + P）で「Export current file as HTML」などを選択

    3. 保存場所を指定して .html で出力

👉 HTMLファイルが生成され、ブラウザで開けば整形されたテキスト表示になります。

✅ 方法②：外部エディタ「Typora」で開いてHTML保存

🔧 Typora（Markdownエディタ）

    • Obsidianと同じ.mdが開けて、「HTMLとして保存」できる超軽量ソフト

    • 日本語もきれいに表示され、Obsidianに近い見た目

▶ 入手先：https://typora.io/

🧰 使い方：

    1. .md ファイルをTyporaで開く

    2. メニュー → 「ファイル → HTMLとして書き出す」

    3. .html ファイルが出力される

✅ 方法③：Pandocでコマンド変換（上級者向け）

pandoc input.md -o output.html

    • 最も柔軟で強力な方法（LaTeX・Word・PDFにも対応）

    • ただしコマンドライン操作が必要

👉 Pandocを使いたい場合はインストール方法＆使い方もサポートできます。

📄 書式整えた状態で出したい Typora（外部）
⚙ 自動処理・一括変換したい Pandoc（上級者向け）

✅ 方法①：Webpage HTML Export（コミュニティプラグイン）

    • Obsidian公式コミュニティプラグインで、Vault全体または単一ファイルのHTML出力が可能です
      obsidian.md+15forum.obsidian.md+15forum.obsidian.md+15

    • インストール手順：

          1. Obsidian → ⚙️「設定」→「コミュニティプラグイン」→ Safe mode：オフ

          2. 「Browse」で Webpage HTML Export を検索

          3. インストール → 有効化（Enable）

    • 使用方法：

        • ノートを開いて → 右クリック → 「Export to HTML」で1ファイル化

        • またはVault全体の出力プラグイン画面で「Multi-file website」等を選んで出力
          github.com
          github.com+8nathang.dev+8forum.obsidian.md+8

✨ 特長：テーマ・CSS・画像・目次も含んだほぼそのままのWebページ形式HTMLが得られる。

✅ 方法②：Export to HTML（by kalvn）

    • 単一のノートをHTMLファイル化 or クリップボードにHTMLコピーできる軽量プラグイン
      obsidianstats.com+2nathang.dev+2reddit.com+2
      github.com+1obsidianstats.com+1

    • インストール手順：
      同様に「コミュニティプラグイン」→ Browse → Export to HTML を検索・インストール・有効化

    • 使用方法：

        • Ctrl + P → Export to HTML: Copy to clipboard as HTML で整形後HTMLコピー可能

        • または ...: Download as HTML でファイル出力されます

✨ 特長：単一ノートのHTML化が超簡単。コピーしてサクラエディタやWordにも貼れる。

✅ 概要：Webpage HTML Export とは？

Obsidianで作成したノート（.md）を、見た目そのままHTMLに変換して出力できるプラグインです。

    • 画像・スタイル・目次も含んだHTML

    • 単一ノート or Vault全体を出力可能

    • 通常のWebページのようにブラウザで閲覧できる

🛠️ステップ①：インストール

    1. Obsidianの⚙「設定（Settings）」を開く

    2. 左の「コミュニティプラグイン（Community plugins）」を選ぶ

    3. Safe Mode を オフ にする

    4. 「Browse」ボタンをクリック

    5. 検索欄に Webpage HTML Export と入力

    6. プラグインが見つかったら → Install（インストール） → Enable（有効化）

🛠️ステップ②：基本の使い方（1ファイル書き出し）

✨ 単一ノートをHTMLとして書き出す：

    1. .mdファイルを開く

    2. Obsidian右上の「…（3点アイコン）」をクリック

    3. Export to HTML を選ぶ（なければ右クリックメニューにもある）

    4. 保存先を選ぶ → .html ファイルとして保存される

📝 生成された .html をブラウザで開くと、Obsidianのテーマが反映された見た目になります！

🛠️ステップ③：Vault全体を書き出す

📦 Vault全体 or 複数ファイルをHTML化：

    1. コマンドパレット（Ctrl+P）を開く

    2. Webpage Export: Export all notes などのコマンドを選択

    3. 出力先を指定すると、フォルダ単位でHTMLと画像・スタイル込みで書き出されます

⚙ オプション設定（必要に応じて）

    1. 設定 → Webpage HTML Export にオプションが表示されます

    2. 変更できる項目例：

設定 内容
Export location 書き出し先フォルダの指定
Include backlinks バックリンク（ParseError: KaTeX parse error: Undefined control sequence: \[ at position 1: \̲[̲note名]）の出力可否
Use custom CSS Obsidianのテーマとは別にCSSを指定するか
Table of contents 自動目次の表示

✅ HTMLの出力結果はどう使える？

    • ブラウザでそのまま読める読み物になる

    • サクラエディタにコピペすれば、書式のない本文として使える

    • OBSで画面キャプチャして縦スク動画に使える
