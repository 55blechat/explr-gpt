✅ Inkscapeが新しいフォントを認識しないときの対処法

1. 🌀 PC再起動がまず最優先

    • フォント（例：あんずもじ2020）をインストールしても、Inkscapeは起動時にしかフォントリストを読み込まない

    • → PCを再起動 or Inkscapeを完全終了→再起動が必要です

2. 🔎 フォントが正しくインストールされているか確認

    • Windowsなら：

      • スタートメニューで「フォント」検索

      • 設定 → フォント 画面に「あんずもじ2020」が表示されていればOK

    • .ttf（TrueType Font）形式であればInkscapeでも問題なく扱えるはず

3. 🔎 Inkscapeでのフォント検索時の注意点

    • 検索時に「moji」など一部しか入れないと出てこないこともある

    • 正式名称（例：AnzuMoji 2020 Regular）で探すとヒットしやすい

4. 🛠️ それでも出てこない場合の対処

対処法 内容
Inkscapeの設定リセット 設定ファイルが壊れている可能性（~/.config/inkscapeを初期化）
別のフォントでテスト 他に最近入れたフォントがInkscapeに出ているか確認
Inkscape再インストール ごく稀にフォントライブラリとの連携に不具合が出ることがある

✅ まず確認させてほしいポイント（順にチェック！）

1. 本当に「TrueTypeフォント（.ttf）」をインストールしましたか？

    • Inkscapeは .otf より .ttf に強いです

    • あんずもじの「2020」版でも .ttf が含まれていることを再確認してください

📂 インストールしたフォントファイルの拡張子は？
→ 例：「AnzuMoji-2020.ttf」ならOK

2. インストールしたフォントが「全ユーザー用」になっているか？

Inkscapeはユーザー単位でインストールされたフォントを読み込めないことがあります

📌 解決法：

  1. 一度フォントを削除

  2. フォントファイルを右クリック → 「すべてのユーザーに対してインストール」

※「インストール」だけでは足りない場合があります（特にWindows）

3. フォント名がInkscape内で「日本語表記」になっていないか？

フォント一覧で「あんず」「anzu」「moji」「2020」「regular」など、部分検索をいろいろ試してください
→ まれに「フォントファイル名」と「表示名」が一致していないことがあります

4. Inkscapeのフォントキャッシュ破損の可能性

    • フォントキャッシュの再構築が必要な場合があります

🛠️ 対処方法（Windowsの場合）：

  1. C:\Users\[ユーザー名]\AppData\Local\Microsoft\Windows\Fonts
    → ここに重複フォントや壊れたフォントがないか確認（削除）

  2. Inkscapeを一度セーフモードで起動（--safe-mode オプション）

  3. または以下のキャッシュ削除を実施：

      • C:\Users\[ユーザー名]\AppData\Local\fontconfig を削除

      • PC再起動 → Inkscape再起動

🧪 最終手段（別ソフトでテスト）

  1. メモ帳やWordで「あんずもじ2020」が選べるか確認

  2. Canvaなど別アプリでフォントの認識があるか試す（アップロード機能あり）

→ 他では見えるがInkscapeだけダメなら、Inkscapeの設定 or キャッシュに問題があると確定できます

📘 文字の背景色

🙂 タケ：
Inkscape 文字の背景色はどこで設定する？


🤖 チャットさん：
Inkscapeでは、文字（テキスト）に直接「背景色」を設定する機能は標準では存在しません。
ですが、“それっぽく見せる方法”はあります。以下にわかりやすく説明します👇

✅ 文字の背景色をつける方法（擬似的）

方法①：文字の後ろに矩形（四角形）を配置する

これが最も一般的で簡単です。

🔧 手順：

  1. テキストツールで文字を入力

  2. 四角形ツール（F4キー）で文字の背後に合わせて矩形を描く

  3. 矩形の塗り色を設定（背景色にしたい色）

  4. メニュー「オブジェクト → 配置」または右クリック → 「最背面に移動」

      • → テキストの下に矩形がくるように

💡ポイント

    • テキストと矩形を選んで Ctrl + G（グループ化）すれば一体化して扱える

    • 角を丸めたいときは、矩形を選択してハンドルをドラッグ

方法②：文字にハイライト風のパスを描く（上級）

    • 複数行テキストで一文字ずつ背景をつけたいときなどに有効

    • パスやクローンを使った背景風演出が可能（凝った表現用）

🎯 まとめ

やりたいこと 方法
単純な背景をつけたい ✅ 四角形を後ろに敷くのが一番簡単で確実
複雑な装飾や自由な背景 パスやエフェクトでカスタム演出（上級向け）

Canvaのように「テキストボックスに背景色」みたいな機能はInkscapeにはないですが、
工夫すればかなり柔軟に演出できます！


📘 線と文字の重なり

🙂 タケ：
ラインと文字が重なってて、そこは文字優先でライン途切れてほいいんだけど。


🤖 チャットさん：
その「ラインと文字が重なっている部分だけ、線が途切れてほしい（文字が優先で見える）」という処理、
Inkscapeでは 「マスク」や「パスの差分（切り抜き）」 を使うことで実現できます。

以下におすすめの2パターンを紹介します👇

✅ 方法①：テキストを最前面に置いて線を分割して隠す（簡単・非破壊）

🔧 手順（簡易版）

  1. テキストを置く

  2. 線（ライン）を選択して右クリック→複製（Ctrl+D）

  3. 複製した線とテキストを選択

  4. メニュー「パス → 分割（Cut Path）」で切り抜き

  5. 切り抜いた線の断片を削除 or 非表示にする

  6. テキストは最前面（PageUpキー）に移動しておく

💡 この方法は、「線に“空白”を作って文字をくっきり見せる」イメージ

✅ 方法②：テキストの形で線を「くり抜く（差分）」

※ 完全な切り抜きになるため、文字の下の線が実際に消える方式

🔧 手順（破壊的・でも精密）

  1. テキストをパスに変換（パス → オブジェクトをパスへ（Shift+Ctrl+C））

  2. 線を複製（Ctrl+D）

  3. 複製した線とパス化したテキストを選択

  4. メニュー「パス → 差分（Ctrl+-）」を適用
    　→ 文字の形でラインがくり抜かれる

  5. 文字を前面に戻す（必要に応じて）

✅ 完成後の線は「穴が空いた状態」になります（再編集しにくいが見た目は完璧）

📝 どちらがおすすめ？

用途 方法
あとから編集したい／とりあえず見せたい ✅ 方法①（分割＆非表示）
しっかり仕上げたい／印刷や書き出し用途 ✅ 方法②（差分でくり抜く）

✅ おさらい：手順まとめ（背景色付きテキスト）

  1. テキストツールで文字を入力

  2. 四角形ツール（F4キー）で矩形を描く

  3. 矩形を塗りたい色に設定

  4. 矩形を最背面に移動（PageDownまたは右クリック → 最背面へ）

  5. テキストと矩形を両方選択して Ctrl+G でグループ化（ずれにくくなる）

💡 便利な調整ポイント

調整内容 操作方法
背景を少し大きめにしたい 矩形サイズを手動で広げる（テキストに合わせて調整）
背景の角を丸めたい 矩形を選んで、角のハンドルをドラッグ（画面上で直接）
複数行の背景をまとめたい 矩形サイズを行数に合わせて調整し、中央揃えにする
